# Factorial-of-a-large-number
## How to use
1. Download 2 files *FactorialCalculator.jar* and *RUNME.bat* [here](https://github.com/duyngnth/Factorial-of-a-large-number/tree/executable) and put them in the same directory
2. Run the file *RUNME.bat*
## Others
This code could also be used to calculate any multiplication of positive numbers, such as the power of large numbers.
